# 편의점 포스기 프로그램

포스기 프로그램



## 분석

포스기 프로그램이 필요하다.

상품 품목을 관리하고 싶다.

- 재고관리는 하지 않는다.
- 상품 추가에 상품명과 가격은 필수다.

주문을 관리하고 싶다.

- 주문 전체 취소가 가능했으면 좋겠다.
- 더치페이로 결제할 수 있으면 좋겠다.

결제 기능이 필요하다.

- 할인 금액도 확인하고 싶다.
- 결제는 카드결제만 가능하다.
